# マウスやキーボードなど、現在の入力情報を可視化するツール

## UI

- ModernWPF を使用する。
- メインウィンドウは、マウスやキーボードの入力を見やすい形で可視化する。
- 例えば、マウス座標についての情報は `(x, y)` のように表示する。
- キーボードは今現在押されているキーのみテキスト等で表示する。


## その他

- 「Always on Top」トグルでウィンドウを最前面に出来るようにする。
- ウィンドウサイズ固定ボタンで出来るだけ小さいサイズに固定できるようにする。
